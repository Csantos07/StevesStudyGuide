## Git

Common Git Commands:
`git init`
`git add <filename>`
`git commit -m "message"`
`git checkout`
`git checkout -b`
`git status` - check the state of the git process
`git pull origin <branch>`
`git push origin <branch>`
if forked:
`git pull upstream <branch>`
`git log`
